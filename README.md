## 👩‍💻 About This Repository

Mobile applications distributed on platforms like the Google Play Store and Apple App Store are required to provide publicly accessible legal documentation regarding user data privacy and terms of usage. **DietraScan Legal** provides a transparent, centrally hosted location for these required static pages, ensuring compliance with app store regulations and data protection laws.

---

## 🚀 Key Features

* **Privacy Policy (`privacy.html`):** Clearly outlines how user dietary data, account information, and usage metrics are collected, stored, and protected within the DietraScan app.
* **Terms of Service (`terms.html`):** Defines the rules, guidelines, and legal agreements between DietraScan and its users.
* **Lightweight & Fast:** Pure HTML/CSS with zero dependencies for lightning-fast loading times.
* **Easily Hostable:** Designed specifically to be hosted seamlessly and reliably via GitHub Pages.

## 🚀 Quick Start / Local Viewing

Because these are static HTML files, no complex server setup is required.

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/hamzabadshah10/dietrascan-legal.git
   ```

2. **Navigate & Open:**
   * Open the `dietrascan-legal` folder.
   * Simply double-click `privacy.html` or `terms.html` to view them locally in any browser.
